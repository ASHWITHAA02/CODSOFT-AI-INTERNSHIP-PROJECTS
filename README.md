# 🤖 CODSOFT AI Internship Projects

This repository contains the projects completed as part of the **AI Internship Program at CodSoft**.
The projects demonstrate practical implementation of **Artificial Intelligence, Machine Learning, and Computer Vision concepts using Python**.

Each project focuses on solving real-world problems using AI techniques such as rule-based systems, game AI, image captioning, recommendation systems, and face detection.

---

## 📂 Project List

### 🧠 Task 1 – AI Chatbot

A simple rule-based chatbot that interacts with users through the command line.

**Features**

* Responds to common greetings
* Handles simple conversational queries
* Lightweight and beginner-friendly implementation

**Technologies Used**

* Python
* Basic Natural Language Processing logic

---

### 🎮 Task 2 – Tic Tac Toe AI

An intelligent Tic Tac Toe game where the player competes against an AI.

**Features**

* Human vs AI gameplay
* AI uses decision logic to choose optimal moves
* Command-line interface for interaction

**Technologies Used**

* Python
* Game logic algorithms

---

### 🖼 Task 3 – Image Captioning AI

An AI model that generates captions for images automatically.

**Features**

* Accepts an image as input
* Generates a descriptive caption
* Uses a pre-trained deep learning model

**Technologies Used**

* Python
* Hugging Face Transformers
* PyTorch
* Computer Vision

---

### 🎬 Task 4 – Movie Recommendation System

A content-based recommendation system that suggests movies based on user preferences.

**Features**

* Recommends movies similar to user input
* Uses similarity algorithms for recommendations
* Works with a movie dataset

**Technologies Used**

* Python
* Pandas
* Scikit-learn
* Machine Learning concepts

---

### 👁 Task 5 – Real-Time Face Detection

A computer vision project that detects human faces in real-time using a webcam.

**Features**

* Live webcam detection
* Face bounding boxes displayed on screen
* Real-time processing

**Technologies Used**

* Python
* OpenCV
* Haar Cascade Classifier

---

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/ASHWITHAA02/CODSOFT-AI-INTERNSHIP-PROJECTS.git
```

Navigate to the project folder:

```bash
cd CODSOFT-AI-INTERNSHIP-PROJECTS
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶️ Running the Projects

Example:

```bash
python chatbot.py
```

Each project folder contains the required Python script to run the application.
